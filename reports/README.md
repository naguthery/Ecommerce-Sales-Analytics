# Sales Performance Analytics Reports

This folder contains the reports created for the Sales Performance Analytics Project. These reports document the data cleaning process, analysis methodology, and key findings related to revenue trends, product performance, customer purchasing behavior, geographic sales distribution, and payment method preferences.

## Contents

* **Sales Performance Analytics Report.pdf** – Comprehensive report detailing the project objectives, methodology, analysis, visualizations, and key findings.

* **Sales Performance Case Study.pdf** – Condensed version of the Sales Performance Analytics Report that highlights the business problem, analytical approach, key insights, and recommendations.

* **Sales Performance Data Cleaning Report.pdf** – Report documenting data quality issues identified in the dataset, the cleaning procedures performed, and the resulting improvements to data accuracy and consistency.
